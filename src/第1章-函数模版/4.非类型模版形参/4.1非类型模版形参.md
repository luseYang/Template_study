# 非类型模版形参
既然有接受“类型模板”，也有非类型的，意思就是模版不接受类型，接收对象或值。  
```cpp
template<std::size_t N>
void f() { std::cout << N << '\n'; }

f<100>();
```
非类型模版形参有众多的规则和要求，目前，只需要认为参数是一个“常量”。  
非类型模版形参也可以有默认值：
```cpp
template<std::size_t N = 100>
void f() { std::cout << N << '\n'; }

f();     // 默认      f<100>
f<66>(); // 显式指明  f<66>
```
