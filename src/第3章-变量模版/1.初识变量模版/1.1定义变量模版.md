# 定义变量模版
和函数模版，类模板一样，变量模版也不是变量，只有在实例化变量模版的时候，编译器才会生成实际的变量  

变量模版实例化之后，简单来说就相当于一个全局变量，所以不用担心它的生命周期  
```cpp
template<typename T>
T v;
```
就很简单，这样就定义了一个变量模版。  
既然叫变量模版，那肯定可以加一些修饰变量的关键字，比如```cv```限定，比如 ```constexpr``` ,当然也可以有初始化器，比如 ```{}``` 、```= xxx```。等  
```cpp
template<typename T>
constexpr T v{};
```
